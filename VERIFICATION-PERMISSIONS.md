# דוח בדיקת הרשאות — בעלים, מנהלים, משתמשים

## סיכום — מה כל תפקיד רואה

| תפקיד | תפריט | נתונים | העלאה |
|-------|-------|--------|-------|
| **בעלים** (owner / isSystemOwner) | הכל + פאנל מנהל | קטלוג גלובלי + כל המסעדות | למסעדה או לקטלוג גלובלי |
| **מנהל** (manager / admin) | הכל + פאנל מנהל | רק המסעדה שלו | רק למסעדה |

| **משתמש** (user) | לפי הרשאות | רק המסעדה שלו | רק למסעדה |

---

## פירוט לפי תפקיד

### בעלים (isSystemOwner או owner)
- **תפריט:** לוח בקרה, עץ מוצר, פאנל מנהל, רכיבים, עלויות תפריט, ספקים, מלאי, הזמנות ספקים, העלאה, דוחות, הגדרות
- **מסעדות:** רואה את כל המסעדות (מחליף ביניהן)
- **התחזה:** רק isSystemOwner — יכול להתחזה כמסעדה
- **נתונים:** רכיבים גלובליים + רכיבי המסעדה, מתכונים, מלאי, ספקים
- **העלאה:** מחירון — אפשרות "שמור לקטלוג הגלובלי"

### מנהל (manager / admin)
- **תפריט:** כמו בעלים (כולל פאנל מנהל)
- **מסעדות:** מסעדה אחת בלבד
- **נתונים:** רק רכיבים ומתכונים של המסעדה שלו
- **העלאה:** מחירון — רק למסעדה
- **פאנל מנהל:** הזמנת משתמשים, הגדרת הרשאות

### משתמש (user)
- **תפריט:** לוח בקרה, עץ מוצר, רכיבים, מלאי, ספקים, הזמנות ספקים, העלאה, דוחות (אם canSeeReports), עלויות תפריט (אם canSeeCosts), הגדרות (אם canSeeSettings)
- **פאנל מנהל:** לא מוצג
- **נתונים:** רק רכיבים ומתכונים של המסעדה שלו

## דפים וסינון נתונים

| דף | בעלים | מנהל | משתמש |
|----|-------|------|-------|
| לוח בקרה | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |
| עץ מוצר | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |
| רכיבים | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |
| עמוד מתכונים | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |
| עלויות תפריט | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |
| מלאי | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |
| ספקים | גלובלי + מסעדה | מסעדה בלבד | מסעדה בלבד |

| הזמנות ספקים | לפי restaurantId | לפי restaurantId | לפי restaurantId |
| פאנל מנהל | גישה מלאה | הזמנה + הרשאות | אין גישה |
| הגדרות | גישה מלאה | גישה מלאה | אם canSeeSettings |

## הגנות בקוד

1. **app/page.tsx:** `canAccessPage` — דוחות, עלויות תפריט, הגדרות, פאנל מנהל (לפי הרשאות)
2. **AdminPanel:** בדיקה `userRole === "user"` → הודעת "אין הרשאה"
3. **נתונים:** `isOwnerRole` — טעינת קטלוג גלובלי רק ל־owner או isSystemOwner
4. **העלאה:** `canSaveToGlobal` — רק לבעלים
